Conda 在 Linux 中的環境配置

  • 第一步,输入命令,打開文件>
    sudo vim /etc/profile


  • 第二步,在最后一行加入
    export PATH=/home/develop/anaconda3/bin:$PATH
    其中”/home/develop/anaconda3”是我的anacond3安装路径,请根据自己的安装路径修改路径目录。


  • 第三步,按ESC鍵,輸入 :wq! 保存退出编辑,終端输入以下命令生效
    source /etc/profile
