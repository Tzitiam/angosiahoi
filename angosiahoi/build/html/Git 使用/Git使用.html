Git使用


下載/拉取


  • 下载（拉取）云库：git clone 网址；

  • 下载（拉取）云库分支：git clone -b 分支名 网址；

初始化项目：git init；

常用操作:

  • 所有文件添加到暂存区：git add . ；

  • 查看命令：git status；

  • 提交文件到本地库：git commit -m ‘提交信息’；

  • 若本地库没有连接到云仓库：连接：git remote add origin 网址 ；

  • 上传文件到云库：git push -u origin “master” ；

若有修改，查看修改了啥：git diff 文件名 ；
